T2 HTML, CSS, and DOM Part 1 Blog

Can you think of an analogy to describe HTML, CSS and the DOM? How would you describe it to your non-technical friend?

HTML is the basic technical side to coding - like a foundation or something to build off. CSS is the design and decorative side to coding, where you're able to make the foundation/base look appealing and creative. the DOM is like these two together, where all of the HTML and CSS is found.

What is meant by boxifying design?

Boxifying design is basically an invisible grid structure on your website so you can align different design elements to make them fit together on the page for the overall look.

What is the box model?

Each element on a website is in a box. The box is made up of Content (text), Padding (the transparent area that surrounds the content), Border (border around the content and padding), and Margin (thetransparent area around the border).

T2 HTML, CSS, and DOM Part 2 Blog

What is grid based design??

Grid based design is a useful tool to organise and structure your website, based on rows and columns. There are 12 columns on a page and a row with the width of 100%.

What is the hype about responsive webpages??

Nowadays, with more people using smart phones, and tablets etc; responsive webpages are a must have. A responsive webpage can be viewed on different screen sizes or and is adaptive to any browser it is being viewed on. The webpage is formatted in a way that when viewing on smaller or bigger screens, the webpage is able to adapt.

What is semantic structure?

A semantic structure is the foundation of your code - a template. It helps the web browser define what the layout of the content is on the webpage - whether its an image, text, or a heading etc.

T2 HTML, CSS, and DOM Part 1 Blog

Can you think of an analogy to describe HTML, CSS and the DOM? How would you describe it to your non-technical friend?

HTML is the basic technical side to coding - like a foundation or something to build off. CSS is the design and decorative side to coding, where you're able to make the foundation/base look appealing and creative. the DOM is like these two together, where all of the HTML and CSS is found.

What is meant by boxifying design?

Boxifying design is basically an invisible grid structure on your website so you can align different design elements to make them fit together on the page for the overall look.

What is the box model?

Each element on a website is in a box. The box is made up of Content (text), Padding (the transparent area that surrounds the content), Border (border around the content and padding), and Margin (thetransparent area around the border).

T2 HTML, CSS, and DOM Part 2 Blog

What is grid based design??

Grid based design is a useful tool to organise and structure your website, based on rows and columns. There are 12 columns on a page and a row with the width of 100%.

What is the hype about responsive webpages??

Nowadays, with more people using smart phones, and tablets etc; responsive webpages are a must have. A responsive webpage can be viewed on different screen sizes or and is adaptive to any browser it is being viewed on. The webpage is formatted in a way that when viewing on smaller or bigger screens, the webpage is able to adapt.

What is semantic structure?

A semantic structure is the foundation of your code - a template. It helps the web browser define what the layout of the content is on the webpage - whether its an image, text, or a heading etc.

A responsive webpage can be viewed on different screen sizes or and is adaptive to any browser it is being viewed on. What mobile first design is, and why it's important.

Mobile first is an approach to responsive design: design for smaller screens first, then add more features and content for bigger and bigger screens. This design approach is also known as "progressive enhancement."

T4 Javascript Basics Blog

How does JavaScript compare to HTML and CSS?

JavaScript works alongside HTML and CSS to basically make a website come alive in a sense - it allows you to add cool features and widgets etc, unlike CSS which is just styling.

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

Control flow is the order in which the functions and/or instructions are undertaken. JavaScript carries this out as it is written. Loops are a segment of the code that is continously repeated (hence the name loop) until an action is completed.

Explain the difference between accessing data from arrays and objects.

Data that is stored in arrays are organised in a list-like structure. In order to access data from an array you can use the position of the list like a 'key', with the stored data being the 'value'. Objects are similar to lists, but instead they store data and functions. In order to access the data in an object you use key-value pairings using a variable name assigned by you.

Explain what functions are and why they are useful.

Functions are useful because they not only accept data, and undergo an action that the data instructs, it will also return a result.

.map()

When the .map function is called on an array it creates a new array made up of the results of a chosen function on each element of the parent array.

.filter()

Creates a new array of all the elements in the parent array which comply to the conditions set out in the function within the filter parameters.

.reduce()

Reduces the values of an array to a single value. It uses two parameters and then a function to return the single value. This can also be used to flatten an array of arrays into a single array.

T6 Javascript Language Blog

In the JavaScript language, describe all the use cases for each of:

Parentheses () - e.g. '1. Execute a function var result = doSomething()':

Parentheses are most commonly used when you are wanting to pass information into a function, or to provide information to loops (for, while) and conditionals (if, else if).

Brackets []:

Square brackets are used to index (access) elements in arrays and also Strings.

Braces {}:

The curly braces are used to define the start and end of the function, they also separate code into blocks within the function. The curly braces help JavaScript to understand the structure of our script and what we want it to do. They are used with many different things including classes, functions, loops, and conditionals.

single quotes '':

JavaScript adopted the use of single quotes in code, allowing for the insertion of double quotes on attributes of appended HTML snippets. But there isn't much difference between the use of Single and Double quotation marks.

double quotes "":

HTML adopted using double quotes in HTML attributes. You can use either single or double. When referencing attributes and strings in code we use quotation marks.
